# Security Policy Sample

## Password Policy

Employees must use passwords that are at least 14 characters long. Passwords must not be reused across company systems.

## Multi-Factor Authentication

Multi-factor authentication is required for all administrative systems, cloud dashboards, source code repositories, and production infrastructure.

## API Key Handling

API keys, access tokens, private keys, and credentials must never be committed to source control. Developers must use environment variables or approved secret management systems.

## Incident Reporting

Security incidents must be reported to the security contact within 24 hours of discovery.
