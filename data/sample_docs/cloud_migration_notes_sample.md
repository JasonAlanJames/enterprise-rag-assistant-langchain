# Cloud Migration Notes Sample

## Migration Strategy

The recommended cloud migration approach is to begin with assessment, then move to pilot migration, phased production migration, validation, and optimization.

## Security

Cloud workloads must use least-privilege IAM permissions. Public access should be disabled unless explicitly required.

## Monitoring

Production workloads must include logging, metrics, uptime monitoring, and alerting.

## Backups

Critical systems must use automated backups and tested recovery procedures.
